# Malak Alnajjar - Hybrid Engineer Portfolio

A creative, modern, and non-traditional portfolio website showcasing the intersection of Telecommunication Engineering and AI & Data Science.

## üéØ Concept

This portfolio is designed as a "Smart Network System" where:
- **Telecom** = the infrastructure (signals, fiber, routing)
- **AI & Data** = the intelligence layer (optimization, prediction, analytics)

The design metaphor is subtly integrated throughout the layout, section naming, and micro-interactions.

## üöÄ Features

### Core Sections

1. **Hero Section** - Animated network background with hybrid identity messaging
2. **Hybrid Timeline** - Unified career journey showing how telecom and AI experiences interconnect
3. **Network & Communication Systems** - Telecom projects with engineering focus
4. **AI-Driven Intelligence Layer** - Data science projects with model/method/output structure
5. **Fusion Projects** - Creative intersection where telecom meets AI
6. **Certifications & Badges** - Interactive certification cards with hover effects
7. **Skills Map** - Visual representation showing overlap between Telecom & AI
8. **Leadership & Impact** - IEEE involvement, volunteering, and recommendations
9. **Download CV** - Easy access to download resume in PDF format
10. **Contact** - Professional contact information and links

### Design Highlights

- Clean, minimalist light theme with white background
- Subtle blue gradient accents
- Smooth animations using Framer Motion
- Responsive design for all devices
- Typography-first design with generous spacing
- Subtle mouse-follow glow effect

## üìù Customization

All portfolio data is stored in `/data/portfolio.ts`. Update this file to customize:
- Timeline experiences
- Projects (Telecom, AI)
- Certifications
- Skills
- Leadership information

### Adding Your CV

To enable the CV download feature:
1. Create a `public` directory in the project root (if it doesn't exist)
2. Place your CV PDF file named `Malak_Elnagar_CV.pdf` in the `public` directory
3. The download button will automatically work

### Updating Contact Information

Edit `/components/Contact.tsx` to update:
- Email address
- LinkedIn profile URL
- GitHub profile URL
